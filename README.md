## 마음고리 클라이언트 앱.

https://apps.ionic.io/apps

###아이폰 빌드
```
ionic platform add ios
ionic build ios
ionic emulate ios
```

###서버 실행
```
ionic serve
```

###앱 업로드
```
ionic login
ionic upload
```
